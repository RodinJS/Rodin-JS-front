How it works?

  1. Log in or Register
  2. Click create new project button
  3. Select one of the templates and fill all fields
  4. Click on project thumbnail to enter the editor mode
  5. Now you can easily manage your project.
  6. If you want to build, publish or make project public, click on a gear for entering project settings
  7. For making project public please click on public toggle
  8. For publishing the project (only premium) please click publish button
  9. For building your first android project go to android tab and fill all fields then click build
  10. Fill keystore and alias password in popup then click build
