Improving SEO mechanisms on Tilda

Redirecting to www. Redirect from www.site.ru to site.ru or from site.ru to www.site.ru in Analytics and SEO > Tilda Webmaster's Panel. Redirecting from http to https for subdomains on Tilda (example.tilda.ws). If you don't have your own domain but use ample.tilda.ws, you can set up a redirect from http to https (domain owners can set this up on cloudflare).
